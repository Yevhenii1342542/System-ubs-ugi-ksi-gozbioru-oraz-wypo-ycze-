# System-ubs-ugi-ksi-gozbioru-oraz-wypo-ycze-
System zarządzania biblioteką — cyfrowe rozwiązanie do automatyzacji zbiorów i obsługi czytelników. Projekt obejmuje katalog online z wyszukiwarką, konto użytkownika do śledzenia wypożyczeń oraz moduł pracowniczy do zarządzania zasobami. System usprawnia zwroty i powiadamia o terminach oraz zaległościach.
