# System-ubs-ugi-ksi-gozbioru-oraz-wypo-ycze-

## Opis projektu
Aplikacja służy do zarządzania księgozbiorem oraz wypożyczeniami w bibliotece.  
Umożliwia czytelnikom przeglądanie i wypożyczanie książek, a pracownikom zarządzanie katalogiem i użytkownikami.

---
